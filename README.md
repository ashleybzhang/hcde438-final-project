# TIFA — Fashion Style & Brand Discovery

A web app for discovering clothing styles and brands that match your aesthetic. Built with React, Firebase Firestore, and Tailwind CSS.

## Features
- Browse fashion styles and brands
- Search and filter by tags
- Save favorite styles and brands to your profile
- User authentication

## Tech Stack
- React (Vite)
- Firebase Firestore + Authentication
- Tailwind CSS
- React Router
